##开发环境
    1)开发技术：spring+springMVC+mybatis
    2)数据库：mysql<数据库名:hwl002>

##功能说明

* 系统功能
    *  登录功能
    *  注册功能
* 帖子管理功能
    *  发布帖子
    *  能删除自己发布的帖子
    *  能给自己发布的帖子分类<新增>
* 分类功能  <新增>
    *  添加分类
    *  删除分类(如果文章只有一个分类、则变为未分类)
    *  给文章分类 <新增>
* 评论管理(回帖管理)
    *  能在帖子下发布自己的留言
    *  能够删除自己帖子下的留言
    *  能够回复别人的留言
* 搜索功能 <新增>
    *  能够按关键字搜索帖子
* 好友管理 <新增>
    *  添加好友
    *  通过好友请求
* 粉丝管理 <新增>
    *  关注功能
    *  仇晓关注

#####说明
    对好友申请通过、分类管理都在右上角个人中心内进行管理
